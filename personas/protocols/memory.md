# Memory Protocol

You MUST write at least one memory file every session.

## Where
Write `.md` files to `.agent-memory/` in your workspace (symlinked to persistent storage).

## What to Write
- Codebase patterns discovered
- Debugging insights and gotchas
- Architecture decisions made
- Key learnings that help future sessions

## Cross-Agent Knowledge
If a learning benefits other agents, also write to `~/.anc/shared-memory/<topic>.md`.
Keep entries under 2000 chars.
