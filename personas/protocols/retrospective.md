# Retrospective Protocol

After completing meaningful work (when you write HANDOFF.md), also write `RETRO.md`:

```markdown
## What went well
- ...

## What could improve
- ...

## Key learnings (for future sessions)
- ...
```

This is NOT optional for tasks — it's how you build institutional memory.
Skip only for conversations/quick answers where no real work was done.

Your retrospectives are accumulated in shared memory. Future sessions benefit from them.
